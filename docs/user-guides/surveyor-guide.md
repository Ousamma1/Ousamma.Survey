# Surveyor Guide

Quick start guide for field surveyors using the Ousamma Survey System.

## Getting Started

### Login
1. Navigate to `/surveyor-dashboard.html`
2. Enter credentials provided by administrator
3. Complete profile setup

## Conducting Surveys

### Step 1: Select Survey
- View assigned surveys
- Check quotas and deadlines
- Read survey instructions

### Step 2: Start Survey
1. Click "Start New Response"
2. Capture respondent consent
3. Enable location services (if required)

### Step 3: Collect Responses
- Read questions clearly
- Record answers accurately
- Use "Save Draft" for interruptions
- Verify responses before submitting

### Step 4: Submit
- Review all answers
- Capture location (auto)
- Click "Submit"
- Confirm submission

## Using the Mobile App

### Offline Mode
- Surveys downloaded automatically
- Work without internet
- Sync when connected
- Queue indicator shows pending uploads

### Location Tracking
- Enable GPS for accurate location
- Verify address displayed is correct
- Required for some surveys

### Photo Capture
- Use camera for photo questions
- Photos compressed automatically
- Clear, well-lit photos preferred

## Best Practices

### Before Starting
☑ Fully charged device
☑ Stable internet (or offline mode ready)
☑ Review survey questions
☑ Prepare introduction script

### During Survey
☑ Introduce yourself professionally
☑ Explain purpose clearly
☑ Maintain neutral tone
☑ Don't influence responses
☑ Respect privacy

### After Completion
☑ Thank respondent
☑ Verify submission successful
☑ Check quota progress
☑ Report any issues

## Troubleshooting

**Cannot submit survey:**
- Check internet connection
- Save as draft
- Try again when connected

**Location not working:**
- Enable location services
- Grant app permissions
- Move to open area

**App crashed:**
- Reopen app
- Draft auto-saved
- Continue where left off
